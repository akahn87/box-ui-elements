### Examples

**Open**
```js
<Sidebar animation='overlay' expanded={true}>
    <Sidebar.Logo>
        Logo Item
    </Sidebar.Logo>

    <Sidebar.Nav>
        Nav Items
    </Sidebar.Nav>

    <Sidebar.Footer>
        Footer Items
    </Sidebar.Footer>
</Sidebar>
```

**Closed**
```js
<Sidebar animation='overlay' expanded={false}>
    <Sidebar.Logo>
        Logo Item
    </Sidebar.Logo>

    <Sidebar.Nav>
        Nav Items
    </Sidebar.Nav>

    <Sidebar.Footer>
        Footer Items
    </Sidebar.Footer>
</Sidebar>
```
